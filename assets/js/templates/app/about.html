Adapted by Jefrey S. Santos (@jesobreira) from Adam Moore (@minikomi)'s original work (Bootstrap Form Builder) to help take the stress out of writing all that markup to get Bulma forms together.

If you have a problem, or want a specific snippet added please check out the github project. Note, this is only a simple tool so I'm not that keen on adding many features (eg. saving/exporting forms, embedding forms etc.).

Adding snippets is quite simple now (due to the rewrite!) so please don't hesitate to open an issue!
